# SUMI BI

#### 介绍
基于SpringBoot + RabbitMQ + AIGC +（React）的智能数据分析平台。区别于传统BI，用户只需要导入原始数据集，并输入分析诉求，就能够自动生成可视化图表及分析结论，实现降低数据分析的人工成本，提高数据分析效率。
